cremea-maker
====

プログラム

## Description
世界観の作り方で一躍有名になったクレメアさんのアイコンを星の形や色を変更できるよう、クレメアメーカーとして作成。
オリジナルのクレメアアイコンを作成できます。
JavaScriptとBootstrapを使用しています。
クレメアマークはcanvasで描画し、星の数や色変更時に再描画して表示させています。
PCでは右クリックなどで保存はできますが、ボタンで画像を保存できるようにしています。
